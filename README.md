
## Introdução
O objetivo desse projeto é a criação de um e-commerce para venda de canecas e camisetas

## Instalação
Instale as dependências com:
```terminal
yarn install
```
Para iniciar a aplicação, execute o seguinte comando:
```terminal
  yarn dev
```
